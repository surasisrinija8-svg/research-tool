# AI Earnings Call Research Tool

This project is an AI-powered research tool that analyzes earnings call transcripts.

## Features
- Upload PDF transcript
- Extract structured financial insights
- Identify management tone
- Extract key positives and concerns
- Detect forward guidance and growth initiatives
- Uses Groq LLM (Llama 3.1)
- Deployed publicly for evaluation

## Live Demo
https://research-tool-3f5s.onrender.com/test

## Tech Stack
- Node.js
- Express.js
- pdf-parse
- Groq LLM API
- Render Deployment
